Project 3: Image Recognition Pictogram

Description

Image Recognition Pictogram (I will think of a better name) is a game that involves a user drawing a picture of an object in VR and the system generating a picture of what it thinks the user drew. For example if I draw four wheels and a square on top of them, the system may think that the user has drawn a car and will generate a car object for the user to then interact with and play with. This will be a fun game that can keep users entertained for a long time. Perhaps I will narrow it down to one specific subset of images later if this task seems very difficult or time consuming.

Background

When I was thinking about coming up with an idea, I was struggling with a few concepts in my Machine Learning class. So I thought to myself why not incorporate this into my VR class so I could learn some more about both classes at the same time. I saw a lot of examples online about how machine learning can be used for image recognition, and that let me to this idea.

Platform/Software

For this project I will be using the HTC VIVE and Unity 5.6.3. I plan to use Tensorflow’s C++ API for building a classifier that will be able to tell what the user has drawn.
