Project 3: PaintVR

Description

Originally I had thought of calling my project Image Recognition Pictogram and had decided that I would change its name to PaintVR and change its functionality as well. PaintVR will now be a tool that you can use to draw and play mini drawing games.

I have designed it after Google’s TiltBrush. It will consist of three modes. The first being a free drawing mode with different colors and textures. The second mode will be a timed mode in which the user is prompted to draw a specific object and scored based on accuracy. The third mode will be a mini drawing game in which the user is asked to draw a specific object and scored based on accuracy.

Background

When I was thinking about coming up with an idea, I was struggling with a few concepts in my Machine Learning class. So I thought to myself why not incorporate this into my VR class so I could learn some more about both classes at the same time. I saw a lot of examples online about how machine learning can be used for image recognition, and that let me to this idea.

When I sat down to work I realized that programming and building a database and then training data and then creating a way for test data to be interpreted and matched against the classifier (also which I’d have to create) I realized I had too much work on my hands. Therefore, I decided to stick to the idea of drawing objects but with a different twist. That is, I will now create a drawing game that involves object recognition but at a smaller scale.

Platform/Software

For this project I will be using the HTC VIVE and Unity 5.5.4.
