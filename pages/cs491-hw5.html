HW 5: AR Furniture

The pictures above were all taken using the android app called Roomle. The nice thing about this application is that it has a plethora of furniture. The downside, though, is that the application’s UI with the objects is not so great. As you can see from the above pictures, the sofa doesn’t exactly seem natural within the apartment. That’s because it was difficult to arrange it (spatially and scale-wise) to make it seem natural.

Specifically, this application would be great if the objects were made to see more naturalistic.

In the future AR applications like this can be applied to a lot more than just furniture. They can be used for pretty much anything that can mimic reality. What does that mean? Well, let’s say for example I want to buy a pair of shoes that I see online but I wanna see how they would look on me first. Moreover, I want to make sure the shoe fits on me well. So what I can do is have an AR object of the shoe that is available to me to slip on (through augmented reality). Then I can slip it on, see if it would be large on me or small. I can see how it would look; I can pair it with a few outfits, etc. Of course I won’t be able to feel how it actually feels to walk in. Applications like this can help both the consumer as well as the retailer/manufacturer.
